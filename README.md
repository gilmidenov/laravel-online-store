# Laravel 12 & Livewire 3.Разработка интернет-магазина

## Интернет-магазина со всем необходимым типовым функционалом:
### 1) Корзина и все ее основные функции;
### 2) Регистрация и аутентификация пользователей;
### 3) Личный кабинет пользователя;
### 4) Оформление заказа;
### 5) Многоуровневое меню категорий товаров;
### 6) Список товаров выбранной категории;
### 7) пагинация, сортировка, выбор количества показываемых товаров;
### 8) фильтр товаров.

## Админская часть сайта, через которую менеджер сможет управлять функционалом интернет магазина:
### 1) CRUD для категорий товаров;
### 2) CRUD для товаров;
### 3) CRUD для пользователей;
### 4) CRUD для заказов;
### 5) CRUD для фильтров.
